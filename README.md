# Plan Reviewer

VS Code extension to review AI-generated plans with inline comments.

## Features

- View and navigate multi-version plans
- Add inline comments (issue, suggestion, question, approval)
- Generate review prompts for AI assistants
- Approve plans with a single click

## Usage

1. Open the Plan Reviewer panel from the Activity Bar
2. Use **Plan Reviewer: New Review** to start reviewing a plan
3. Click the `+` icon on any line to add a comment
4. Use the **Generate Prompt** button to export comments as an AI prompt
